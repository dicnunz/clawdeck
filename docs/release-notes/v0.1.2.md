# Clawdeck v0.1.2

Distribution and first-run polish release.

## Changed

- Moved the package identity to `@nicdunz/clawdeck` because the unscoped `clawdeck` npm name is already owned by another maintainer and the npm account is `nicdunz`.
- Added public scoped npm publish metadata.
- Added a `prepublishOnly` guard that runs the full verification suite before npm publish.
- Updated launch and README commands to use `npx @nicdunz/clawdeck`.
- Kept the GitHub installer documented as the fallback when the scoped package has not been published yet.

## Fixed

- `clawdeck drill` now detects when it is being run from the Clawdeck source checkout and tells the user to adopt an OpenClaw workspace instead of suggesting source-repo scaffolding.
- Release runbook now matches the already-public GitHub repo and uses `v0.1.2` instead of retagging an existing release.

## Verification

- `npm run verify`
- source-checkout drill regression test
- scoped npm name availability check
