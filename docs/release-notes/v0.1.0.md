# Clawdeck v0.1.0

Initial public release.

## Included

- `clawdeck init` workspace scaffolding for OpenClaw-style command centers
- `clawdeck local` local-only Codex-feeling OpenClaw/Ollama workspace scaffolding
- `clawdeck apply` backed-up local-only OpenClaw profile writer
- `clawdeck audit` local stack score with Markdown, HTML, JSON, and SVG outputs
- `clawdeck drill` no-wifi readiness gates and next-action verdict
- `clawdeck doctor` checks for Node, OpenClaw, gateway, config, and Ollama
- `clawdeck snapshot` redacted public setup export
- local-only OpenClaw template with Ollama primary and no hosted fallback aliases
- README, demo script, product spec, security notes, launch kit, schema, and CI
- source-backed claim check with explicit offline/free/model-quality caveats

## Safety

The snapshot path redacts auth-shaped keys, email addresses, and local home paths. It does not read private OpenClaw agent state, tasks, memory databases, browser profiles, or OAuth profile files.
