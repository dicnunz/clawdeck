# Clawdeck v0.1.5

This is a GitHub source release for the self-serve operator-kit route added after `v0.1.4`.

## Added

- `clawdeck support` now includes the $39 Agent Browser Operator OS route for browser/account/public-action control templates.
- README, package funding metadata, and GitHub funding metadata now surface the same self-serve kit route while preserving the $5 receipt, setup-audit, mini-audit, and full-audit links.
- Package keywords now include browser-ops, approval-workflows, and operator-kit discovery terms.

## Boundaries

- The Agent Browser Operator OS route is positioned as templates for lanes, approvals, proof capture, handoffs, and go/no-go checks.
- It does not fix the Codex Chrome plugin, guarantee browser automation, provide account access, include custom setup, or include calls.
